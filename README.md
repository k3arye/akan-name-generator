# Akan Name Generator
#### This webtool is designed to calculate the user's Akan name using their gender and birthdate.

#### By Esther Kiarie

## Description
Akan names are traditionally of Ghananian culture whereby children are given names based off the day of the week they were born. The Akan name generator calculates the day of the week the user was born and therefore assigns a name to the user.

## Setup/Installation Requirements
* Clone this repository
* Open index.html in your web browser
* Enter your birth date and select your gender
* Click submit to see your Akan name
      
## BDD (Behavior Driven Development)
Input: Birth date and gender
Output: Akan name based on day of birth and gender
